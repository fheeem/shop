
Shop Project
目的

本プロジェクトは、フロントエンド開発の基礎スキル向上を目的として制作したWebサイトです。

HTML / CSS / JavaScript を用いて、ECサイト風のUI構造および基本的なページレイアウトの実装を行いました。
実際のWebサイト構造を意識しながら開発を行い、コンポーネント構造や画面設計の理解を深めることを目的としています。

🛠 使用技術
HTML5
CSS3
JavaScript（Vanilla JS）
🖥 主な機能
商品一覧風レイアウトの実装
ナビゲーションバーの構築
レスポンシブ対応（基本対応）
HTML/CSSによるUIデザイン再現
JavaScriptによる簡単な動作実装
